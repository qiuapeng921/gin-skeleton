# micro-base

```
1. golang 版本 >= 1.18
```

### 配置私有仓库和加速器
```
Bash (Linux or macOS)
export GOPROXY=https://goproxy.cn,direct

PowerShell (Windows)
# 配置 GOPROXY 环境变量
$env:GOPROXY = "https://goproxy.cn,direct"
```

### 初始化项目
```
git clone https://github.com/qiuapeng921/gin-skeleton.git
cd micro-base
go mod tidy
```
